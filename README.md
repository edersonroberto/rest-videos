Esse projeto foi criado com:
* Eclipse IDE for Enterprise Java Developers. Version: 2019-06 (4.12.0)
* Java Versão 8
* Apache Tomcat v8.5 Server


## Setup
Para rodar esse projeto utilizar:o 
1. Eclipse versão 2019-06 ou superior.
